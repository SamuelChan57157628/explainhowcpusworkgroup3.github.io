Introduction

           

The Central Processing Unit (CPU) is an essential physical hardware in a computer system. It performs arithmetic and logical operations, processing data and providing instructions to other components in the computer system. In other words, the CPU serves as the brain of a computer system. Different CPUs have different kinds of architecture, such as x86, CISC, RISC, Arm etc. It controls and affects how the CPU carries out instructions. In the following sections, we will explore further about different types of CPU architectures, how they differ from one another and their strengths and weaknesses. Besides, we would dive into the CPU instruction processing and discover how it processes instructions in each stage.
